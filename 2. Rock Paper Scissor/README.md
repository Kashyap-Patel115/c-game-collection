# ✊✋✌️ Rock Paper Scissors in C

A simple console-based Rock Paper Scissors game where the user plays against the computer.

## 📌 Features

- User vs Computer
- Random computer choice
- Input validation
- Displays winner
- Beginner-friendly code

## 🛠 Technologies Used

- C Programming
- GCC / MinGW
- Visual Studio Code

## 📂 File

- `rock_paper_scissors.c`

## ▶️ Compile

```bash
gcc rock_paper_scissors.c -o game
```

## ▶️ Run

Windows

```bash
game.exe
```

Linux/macOS

```bash
./game
```

## 📚 Concepts Used

- Variables
- if-else
- Random numbers
- User input
- Logical operators
- Standard libraries
