# 🎮 C Games Collection

A collection of beginner-friendly console games developed in C.

## 🛠 Technologies

- C Programming
- GCC / MinGW
- Visual Studio Code

## 🚀 How to Run

Open the desired game folder and compile it.

Example:

```bash
gcc number_guessing_game.c -o game
```

Run:

```bash
./game
```

## 📚 Concepts Covered

- Variables
- Loops
- Functions
- Arrays
- Strings
- Pointers
- Random Numbers
- File Handling
- 2D Arrays
