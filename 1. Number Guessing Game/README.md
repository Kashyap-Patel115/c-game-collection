# Number Guessing Game

A simple console-based Number Guessing Game written in C.

## Features

- Random number generation
- Unlimited attempts
- Hints (Too High / Too Low)
- Counts the number of attempts
- Beginner-friendly code

## 🛠 Technologies Used

- C Programming
- GCC / MinGW
- Visual Studio Code

## 📂 File

- `number_guessing_game.c`

## ▶️ How to Compile

```bash
gcc number_guessing_game.c -o game
```

## ▶️ How to Run

Windows:

```bash
game.exe
```

Linux/macOS:

```bash
./game
```

## 📚 Concepts Used

- Variables
- Loops
- if-else
- Random numbers
- Standard libraries
- User input
